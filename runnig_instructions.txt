1. Requirements
python3
numpy
(pip3)

To install numpy run:

$ pip3 install numpy

Test and training data should be in ./data folder


2. Run parameters estimator

$ python3 parameter_estimator.py

Output files:

full_pooled_cov.txt #for  pooled covariance matrix
full_class_specific_cov.txt #for full class specific covariance matrix
pooled_diagonal_cov.txt #for pooled diagonal covariance matrix
class_specific_diagonal_cov.txt #for diagonal class specific covariance matrix

3. Run classifier

$ python3 classify.py
